Detailed Description

Repositorio de acceso a datos para la entidad Tracking.

Esta interfaz extiende JpaRepository, lo que provee un conjunto de operaciones CRUD (crear, leer, actualizar, eliminar) sobre la entidad Tracking sin necesidad de implementar código adicional.

Además, define consultas específicas relacionadas con el historial de seguimiento de un Envio.

List< Tracking > com.grupo7.tpi.envioslogistica.repository.TrackingRepository.findByEnvioOrderByTimestampAsc ( Envio envio)

Obtiene la lista de registros de seguimiento asociados a un envío específico, ordenados cronológicamente por su marca de tiempo.

Este método aprovecha la convención de nombres de Spring Data JPA para generar automáticamente la consulta SQL correspondiente.
